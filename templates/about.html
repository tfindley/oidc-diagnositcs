{# ── Technology Stack ── #}

Technology Stack

Python 3 + Flask
Web framework — server-rendered Jinja2 templates, no frontend build step
Authlib
OAuth 2.0 and OpenID Connect client library — handles the authorisation code flow and PKCE
Jinja2
Templating engine for server-side HTML rendering
Vanilla CSS + JS
No npm, no webpack, no frameworks — pure CSS variables, CSS grid, and browser-native APIs
Web Crypto API
Browser-native cryptography for JWT signature verification (RS/PS/ES families)
Docker
Container packaging for easy self-hosted deployment

Built with Claude Code — Anthropic's AI coding assistant — using claude-sonnet-4-6 (Claude Sonnet 4.6). The UI, backend, and OIDC diagnostic logic were developed iteratively through AI-assisted pair programming.
